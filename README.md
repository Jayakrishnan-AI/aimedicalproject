# 🩺 Rule-Based Medical Diagnosis System

## Project Overview
This project is a **Rule-Based Medical Diagnosis System** developed using the principles of **Traditional Artificial Intelligence**. The system asks the user about symptoms and suggests possible diseases using predefined medical rules.

The program works by comparing user symptoms with a knowledge base of diseases and symptoms and calculating the best possible match.

This project demonstrates how **expert systems** in traditional AI make decisions using logical rules.

## Features
- Interactive symptom input
- Rule-based disease diagnosis
- Knowledge base of diseases and symptoms
- Multiple disease prediction
- Simple Python implementation

## Technologies Used
- Python
- Rule-based AI logic
- Text-based knowledge base
